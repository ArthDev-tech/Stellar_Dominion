# Building icons

Place PNG images here to show as icons in the planet view building slots:

- **orbital_station.png** – Orbital Station (space station art)
- **small_shipyard.png** – Small Shipyard (small ship art)
- **medium_shipyard.png** – Medium Shipyard (medium ship art)

Use your SpaceStation, SmallShip, and MediumShip art; name the files as above. Icons are optional; if a file is missing, the building still appears with its name only.
